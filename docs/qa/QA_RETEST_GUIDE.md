## Also in this batch (infrastructure — critical)

- **Group-call encryption native module restored and committed.** The
  `BravoFrameCryptorModule` Kotlin module existed only on one build machine and
  was never in git — any fresh build had group calls failing instantly with
  "call failed" (S6 refusal). It was recovered (decompiled from the v1.0.46 APK,
  restored as source, registered, and force-added to git) and verified in a live
  encrypted group call. This unblocks every future build.
- **react-native-webrtc native patch captured** into `patches/` (was loose in
  node_modules — one `npm ci` away from being lost).

## Re-test matrix (minimum)

1. 3 devices, 3 accounts, 3 groups (each account admin of one group).
2. **Audio calls A–H:** every host × every group combination (admin-hosted AND non-admin-hosted). All must connect with 2-way audio.
3. **Group texts between every call** — all must display on all 3 devices.
4. **Video calls:** start as video (camera live at join), all tiles visible on all devices, toggle camera on/off mid-call.
5. **Resilience:** one device toggles airplane mode 5s mid-call → must auto-rejoin. Host hangs up pre-answer → ringing stops everywhere.
6. Report any `tamper`, black tile, ghost ring, or stuck "reconnecting" with device logs (`adb logcat`) and the time it happened.
